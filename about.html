About Us

Learn more about USL and our approach.

Skilled Software Developers

USL are a software company skilled in the development of intuitive software systems and the creation of interfaces to these systems. We combine the best user interface design with robust and secure technology. We deploy these solutions to the Cloud and provide associated technical services.

  • Intuitive User Interfaces

    Our team has over twenty years’ experience of delivering the best in User Interface and User Experience Design. Starting with a Requirements Capture, User Definition, and Discovery Workshop, we work closely with you to ensure we are building the best solution for your Users' needs.

  • Software ‘Best-fit’ Solution

    We will work with you to provide the best software and technology solution for your project. We have a 100% track record of delivering software that works, on time and on budget.

  • Robust, Secure and Scalable Technology

    We use Microsoft Azure Cloud as our ‘go-to’ hosting solution. It is quick to set up, scalable, reliable, and secure. However, we can accommodate your existing hosting solution or work with your preferred supplier.

Discover

Meeting you face to face and making sure we are a good fit for each other and can work together. This is the essential foundation of a successful project.

Define

Through Requirements Capture, User Definition and Discovery Workshops we drill into the detail of the project. Who are the Users? What do they require? How do they require it?

We come back to you with an outline specification for comment and to ensure we understand your requirements.

Design

In advance of an iteration, our User Experience Designer will produce interactive designs, sharing these with you, the client, for feedback. This ensures that the software developers know exactly what they will be creating and the client has a clear understanding of what the software will do.

Develop

USL use an iterative Agile approach to planning and delivering your project. So, in addition to the Specification, the Statement of Work will let you see the project tasks that make up each two-week development phase (Sprint). This also ensures that you have visibility into progress as the project evolves.

Testing

We test each piece of work as the project progresses. We find any bugs and fix them.

We also carry out ongoing User Acceptance Testing (UAT) on the user journeys to ensure the software is intuitive and robust. If we find any issues, we iterate and improve.

Deploy and Launch

Once you have a fully functioning, tested piece of software that we are both happy to launch, we will initiate a soft launch pressure test to ensure the hosting platform can cope with potential user volumes.

  • Unit Tests

    Developers test their own code with Unit Tests. These are automated and added to our build process.

  • Integration Tests

    Developers write Integration Tests to test the process end-to-end for new components. These are added to our release process.

  • Regression Tests

    Regression Tests are formed from the previous Integration Tests and are run to ensure that new code hasn’t affected previous functionality.

  • Automation

    Automation helps us ensure that software builds, releases, and tests are all repeatable and easy to do. This is called Continuous Delivery.
